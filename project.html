Minecraft Server Cluster

Project Facts

DateJanuary 2023
TechnologiesKubernetes, Python, Bash
CodeGitHub
StatusActive

This project created a scalable Minecraft server network using microservices, automatically adjusting resources based on player count.

Technical Details

The system uses Kubernetes to orchestrate server pods, with a custom Python load balancer that:

  • Monitors player counts across instances
  • Automatically scales up/down
  • Maintains persistent world storage
# Example deployment config
apiVersion: apps/v1
kind: Deployment
metadata:
  name: mc-server
spec:
  replicas: 3
  selector:
    matchLabels:
      app: minecraft
  template:
    spec:
      containers:
      - name: mc-container
        image: itzg/minecraft-server
