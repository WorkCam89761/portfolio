System Automation Scripts

Essential utilities for homelab management

DDNS Updater

Function: Automatically updates DNS records when public IP changes

  • Runs hourly via cron scheduler
  • Integrates with Cloudflare API
  • Logs changes to /var/log/ddns.log

Gateway IP Locker

Function: Maintains static network configuration

  • Prevents unauthorized gateway changes
  • Runs at system boot and every 5 minutes
  • Sends alerts via local notification system
