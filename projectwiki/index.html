Homelab Projects

TrueNAS Scale

TrueNAS Scale is the operating system I use to host all my services and containers. To ensure stability:

  • Configured a static IP (192.168.1.69) for the host machine to prevent DHCP conflicts
  • Set up multiple DNS servers (8.8.8.8, 9.9.9.9, 1.1.1.1) for reliable domain resolution
  • Assigned my router (192.168.1.1) as the gateway IP
  • Implemented a shell script to lock these settings and prevent accidental changes

Emby Media Server

Emby is a self-hosted media server that allows streaming of movies, TV shows, and personal videos. Unlike subscription services like Netflix, this gives me complete control over my media library while maintaining privacy.

Immich Photo Backup

Immich serves as a private alternative to Google Photos, automatically backing up photos and videos from my phone. By self-hosting, I keep my personal memories secure rather than storing them on third-party servers.

Cryptocurrency Mining

MMPOS Mining OS

MMPOS is a Linux distribution specifically designed for cryptocurrency mining. Key features include:

  • Web-based rig monitoring and management
  • Telegram notifications for hardware issues
  • More cost-effective than my previous HiveOS setup

GPU Mining Rig

My current setup features:

  • Custom Octominer case (modified server chassis)
  • 12 AMD BC160 GPUs with HBM2 memory
  • Primarily mines Ironfish, with flexibility to switch to Ravencoin
  • Optimized for memory-based algorithms
