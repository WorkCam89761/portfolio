Homelab Infrastructure

Detailed documentation of self-hosted services and configurations

TrueNAS Scale

Primary NAS and container host running on custom hardware with the following configuration:

Host IP
192.168.1.69 (Static)
DNS Servers
8.8.8.8, 9.9.9.9, 1.1.1.1
Gateway
192.168.1.1

Network Configuration Script

#!/bin/bash
# Lock network settings
ip addr add 192.168.1.69/24 dev eth0
ip route add default via 192.168.1.1
echo "nameserver 8.8.8.8" > /etc/resolv.conf
echo "nameserver 9.9.9.9" >> /etc/resolv.conf

This configuration ensures stable networking even if VMs attempt to modify settings. The script runs at boot via cronjob.

  • Automatic phone backup (Android/iOS)
  • Facial recognition
  • Album sharing
  • RAW photo support
