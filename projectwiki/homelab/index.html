Homelab Infrastructure

Detailed documentation of self-hosted services and configurations

TrueNAS Scale

Primary NAS and container host running on custom hardware with the following configuration:

Host IP
192.168.1.69 (Static)
DNS Servers
8.8.8.8, 9.9.9.9, 1.1.1.1
Gateway
192.168.1.1
Storage
5x6TB HDDs (ZFS RAIDZ2)
Cache
2x1TB NVMe SSDs

Key Features

  • Static IP configuration prevents DHCP conflicts
  • Multiple DNS servers for reliable resolution
  • Hosts all Docker containers and VMs
  • ZFS filesystem with data integrity checks

Emby Media Server

Self-hosted alternative to commercial streaming services with these capabilities:

Media Formats
4K HDR, HEVC, AV1
Transcoding
Intel QuickSync + GPU
Users
Family (5) + Friends (3)

Key Benefits

  • Complete ownership of media library
  • No subscription fees or content restrictions
  • Automated metadata fetching
  • Hardware-accelerated transcoding

Immich Photo Backup

Private alternative to Google Photos featuring:

  • Automatic phone backup (Android/iOS)
  • Facial recognition
  • Album sharing
  • RAW photo support

SMB Shares

Network file sharing with these features:

Protocol
SMB 3.1.1
Clients
Windows/Mac/Linux
  • User-based permission controls
  • Fast local network transfers
  • Integrated with TrueNAS permissions
